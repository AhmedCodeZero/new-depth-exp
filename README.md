## Admin CMS

- Set an admin token in a `.env.local` file:

```
ADMIN_TOKEN=your-secure-token
```

- Visit `/admin` to edit content. Paste your token when saving.
- Content is stored as JSON under `content/` and served via `/api/content/[page]`.
